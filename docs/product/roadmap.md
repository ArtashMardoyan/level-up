# Product Roadmap

> **Derived from: Product Model §6 (User journey), §10 (Long-term vision).**

The platform-level, long-term direction. This organizes the vision milestones from the
[Product Model](PRODUCT_MODEL.md); it invents no milestone beyond §6 and §10. Detailed,
phase-by-phase plans live in each feature's own `roadmap.md` under `docs/product/<feature>/`.

## Horizons

**Underway** — the direction already in motion (§10):
- **Realtime voice AI interviews** — the `HTTP → streaming → voice → realtime` path for
  the interview's conversational engine (§7, §10).

**Next**:
- **Adaptive, personalized learning** — driven by each user's per-question performance
  history, which is captured from day one (§5, §10).
- **Full English-communication coaching** — a personalized Dictionary / Learning Profile
  for non-native speakers (§10).
- **Assessment** — the upfront journey stage marked *future* in §6.

**Later**:
- **Mobile** — the ecosystem on phones (§10).
- **Company- and role-specific interview modes** (§10).

## The through-line

Every horizon serves the same arc from §10: Level Up grows from "practice interviews with
an AI" into a **personalized, multi-modal learning ecosystem**. Any new milestone must
first be added to the Product Model (§Governance) before it appears here.
